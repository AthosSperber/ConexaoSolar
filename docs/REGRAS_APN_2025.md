# Regras consolidadas — APN Dezembro 2025

Este documento consolida **regras e condições** extraídas do material APN (Dezembro/2025) para manter o copy do site alinhado.

- O PDF/fonte original **não é versionado** no repositório (ver `docs/REPO_HIGIENE.md`).
- Este arquivo é a “fonte viva” para orientar ajustes de texto em `src/data/productDetails.ts` e nas sections do site.

## iGreen Club (clientes)

### Acesso
- **Todos os clientes iGreen** recebem **acesso gratuito** ao iGreen Club.

### Roleta / números da sorte
- A roleta existe, mas **não é um benefício automático**.
- Regra de mecânica: **1 giro + 1 número da sorte por indicação válida pelo app iGreen Club**.
- Condição: o **cliente indicado** precisa estar **validado e ativo**.

### Cashback
- Cashback (quando aplicável) é usado **para abater na conta de energia**.
- Detalhes/valores/limites dependem de **regras e campanhas vigentes**.

### Link de regras
- Sempre que o site mencionar iGreen Club / roleta / cashback, incluir referência para regras e detalhes:
  - https://www.igreenenergy.com.br/igreenclub

## Telecom
- Todos os planos (Telecom) incluem **iGreen Club grátis**.

## Observação de copy (segurança)
- Evitar promessas absolutas (“ganhe”, “sem limites”, “até X% garantido”) quando depender de condições.
- Preferir: “conforme regras/campanhas”, “quando aplicável”, “mediante validação”, “indicação válida”.
